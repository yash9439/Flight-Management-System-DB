Flight Management System Database

This repository contains the SQL schema and sample data for a Flight Management System database. The database is designed to manage various aspects of airline operations, including information about flights, passengers, reservations, employees, aircraft, maintenance history, locations, feedback, logs, and more.

## Features:

- Structured SQL schema for creating tables related to employees, airplanes, flights, passengers, reservations, maintenance history, and more.
- Sample data population scripts to provide a realistic dataset for testing and exploration.
- Queries and operations for retrieving, inserting, updating, and deleting data in the Flight Management System.
- Aggregate operations to find insights, such as the flight with the highest occupancy.
- Well-defined structure for organized and efficient database management.
